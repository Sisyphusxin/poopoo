# 💩 拉了么（你的健康小工具）

一个 **纯前端、零依赖** 的小小记录工具：今天拉了几次、几点拉的，全都记在 **本机浏览器** 里——**不上传、不登录、不社死**（除非你主动把浏览器借给别人）。

移动端友好，可 **添加到主屏幕** 当个伪 App 用（PWA）。

---

## ✨ 能干什么

- **今日**：一键记一笔，状态文案和表情包会随次数变化（克制地骚一下）。
- **回顾**：月历扫一眼，有记录的日期会有「有故事」的标记；点某天看详情。
- **补拉**：过去哪天忘了记？补上。
- **长按多选删除**：手滑多记了？长按进入多选，底部一键删。

数据格式简单：`localStorage` 里按日期存 `times` 数组——你自己的小黑匣子。

---

## 🚀 怎么跑起来

无需安装 Node，**双击或静态托管** 即可

---

## ⚠️ 人间真实 FAQ

- **换手机/清缓存**：数据跟着浏览器走，**没有云备份**；目前未内置导出。
