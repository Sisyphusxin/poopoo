# 💩 拉了么（严肃健康类·不严肃命名）

一个 **纯前端、零依赖** 的小小记录工具：今天拉了几次、几点拉的，全都记在 **本机浏览器** 里——**不上传、不登录、不社死**（除非你主动把浏览器借给别人）。

移动端友好，可 **添加到主屏幕** 当个伪 App 用（PWA）。

---

## ✨ 能干什么

- **今日**：一键记一笔，状态文案和表情包会随次数变化（克制地骚一下）。
- **回顾**：月历扫一眼，有记录的日期会有「有故事」的标记；点某天看详情。
- **补拉**：过去哪天忘了记？补上（带时间）。
- **长按多选删除**：手滑多记了？长按进入多选，底部一键删。

数据格式简单：`localStorage` 里按日期存 `times` 数组——你自己的小黑匣子。

---

## 🚀 怎么跑起来

无需安装 Node，**双击或静态托管** 即可：

1. 用浏览器打开 **`index.html`**，或  
2. 丢到任意静态服务器 / **GitHub Pages** 上（需 **HTTPS** 才方便 PWA）。

开发调试：`prototype.html` 与主应用功能对齐，标题带「· 原型」方便区分。

## 🌐 部署到 GitHub Pages（懒人版）

把仓库推到 GitHub，**Settings → Pages** 选分支与目录，拿到 `https://<你>.github.io/<仓库>/` 这种地址即可分享。

若仓库在 **子路径**（例如 `/poopoo/`），请确认 `manifest.json` 里 **`start_url` / `scope`** 用的是相对路径（如 `./`），否则「安装」可能跳进错页面——细节见 `docs` 里设计文档。

## ⚠️ 人间真实 FAQ

- **换手机/清缓存**：数据跟着浏览器走，**没有云备份**；要长期留着请自行导出思路（目前未内置导出）。
- **一家人共用一台手机**：是同一份本地数据，**不会**按人自动拆分（除非你每人用自己的浏览器/设备）。
- **名字叫「拉了么」还是页面里「今日拉了吗」**：桌面快捷方式/安装名跟 **manifest / apple 元数据**；页面里空态主文案可以仍是 **「今日拉了吗」**——具体协议以你当前 `index.html` 为准。
